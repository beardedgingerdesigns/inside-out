# Typography

**Status:** suggested in the brief, not finalized.

## Pairing direction

- **Headings:** humanist serif or rounded sans-serif — warm, approachable. Brief suggests *DM Serif Display* for headlines.
- **Body:** clean, highly readable sans — *Inter* or *DM Sans* per the brief.
- **Base size:** 16–18px.

The two-font pairing is non-negotiable per [CLAUDE.md](../../../CLAUDE.md) anti-generic rules: never the same font for headings and body.

## Type rules (from CLAUDE.md)

- Tight tracking on large headings: `letter-spacing: -0.03em`.
- Generous line-height on body: `1.7`.
- Pair a display/serif with a clean sans (the DM Serif Display + Inter/DM Sans pairing fits).

## Avoid

- Anything that reads as corporate or cold.
- Default Tailwind UI font stacks without conscious customization.
- Identical fonts across head and body.

## Open questions

- Is DM Serif Display the final choice, or should we test alternatives (e.g., Fraunces, Source Serif, Lora) for warmth?
- Display weight at hero scale — single weight or variable axis?
