# Mobile Crisis Coverage — 13 South-Central Counties

**Status:** active
**Date:** 2026-05-29
**Scope:** Mobile Crisis coverage lists on mobile-crisis-response.html + partner-providers.html; services/mobile-crisis-response.md

## Decision

Inside Out's Mobile Crisis Response covers **13 south-central Iowa counties**: Adair, Adams, Audubon, Clarke, Dallas, Decatur, Guthrie, Lucas, Marion, Ringgold, Taylor, Union, Wayne. This is the single authoritative list for both the public Mobile Crisis page and the Provider Hub. Inside Out does **not** cover Polk (that's Broadlawns / 911).

## Rationale

Derived from the Iowa BHDS Mobile Crisis Response map (7.25.25) — Inside Out is the yellow region. Melinda confirmed the map shading is accurate (the PDF's text table is not); Justin confirmed 2026-05-29 ("the map is correct," D5). The 13 = the 11 counties in the PDF table **plus** Adams and Union (yellow on the map, omitted from the table), which matches the "13 counties" figure the Provider Hub already cited.
